MiniLang Programming Language
Version 1.0

MiniLang is a small and simple scripting language created for learning
and experimentation. It is designed to be easy to understand,
lightweight, and beginner-friendly.

Core Features:
- Print text to the screen
- Basic math operations (add, sub, mul)
- Variables (set / get)
- Conditions (if)
- Loops (loop)
- Run external .mini files

Example Code:

print "Hello World"
set x 10
get x
loop 3 print "Looping"

File Extension:
.mini
